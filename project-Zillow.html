Description

This project focused on analyzing housing data similar to listings found on Zillow and building a model to predict house prices based on property features. Using Python, I explored trends in housing data, cleaned and prepared the dataset, and compared different models to evaluate prediction accuracy.

The project emphasized understanding which features most strongly influence home prices and how data preprocessing and feature engineering impact model performance.

Skills Learned

  • Python-based data analysis and preprocessing
  • Cleaning and organizing real-world datasets
  • Exploratory Data Analysis (EDA) to identify pricing trends
  • Feature engineering for predictive modeling
  • Building and evaluating models such as Linear Regression and Random Forest
  • Interpreting model results and improving accuracy

Tools Used

  • scikit-learn
