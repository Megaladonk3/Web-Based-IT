Description

This project involved analyzing housing data similar to Zillow listings to understand trends and predict home prices. Using Python, I cleaned and prepared the dataset, explored relationships between features, and built models to estimate housing prices.

The focus of the project was identifying which property features most strongly influence price and comparing different modeling approaches to evaluate performance.

  • Cleaning and organizing real-world datasets
  • Exploratory Data Analysis (EDA)
  • Feature engineering for predictive modeling
  • Building and evaluating models such as Linear Regression and Random Forest
  • Interpreting model results and accuracy metrics
